Demo script (no video required)

1. Open the Vercel URL in Chrome on Android (or localhost dev).
2. Add to Home Screen if you want PWA behavior.
3. On load, demo creates sample memories. Wait ~3s to see geofence trigger appear in the Why panel.
4. Type "pick up" in the search box and press Search. Results should show "Pick up Alex order".
5. Click Create Ephemeral, then Burn Ephemeral. Check AuditPanel to confirm burn manifest logged.
6. Open developer tools (desktop) or remote debugging to confirm no network calls for model inference.
